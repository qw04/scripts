# Scripts

## stopSleep

alt-tab to prevent your computer from sleeping

setup: `pip install -r stopSleep.txt`\
usage: `python stopSleep.py [seconds]`

## pdfRE

Extract information from a pdf using regex that is applied to each paragraph of the pdf

setup: `pip install -r pdfRE.txt`\
usage: `python pdfRE.py [pdf file path]`
